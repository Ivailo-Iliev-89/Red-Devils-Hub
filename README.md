# Manchester United Fan Site

A modern fan website dedicated to Manchester United.
The project focuses on Django REST Framewokr, HTML structure, CSS styling and JavaScript.
This is part of my learning a Full-Stack developer.

## Features

- Homepage with club information
- Styled sections and layout
- Responsive design
- Features highlighting trophies, traditions, and stadium
- Gallery with images of the team, stadium, and trophies
- Players with current squad table (interactive with JavaScript)
- Testimonial section for fans and tickets
- Responsive navbar and footer
- Gradient background and interactive hover effects
- Security settings with dotenv

## Used Techs
- Django
- HTML
- CSS
- JavaScript
- Postgres
